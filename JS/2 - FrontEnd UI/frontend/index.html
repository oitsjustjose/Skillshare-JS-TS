    Why This Isn't Perfect 

    Since the list of notes is just an array, it is only stored in memory. This means that when the server restarts, there's no way to make it last between restarts. We can always read and write it to the drive, but that's not fast, secure or wise (in the event of a power-outage, for example, any new data would likely be gone).
